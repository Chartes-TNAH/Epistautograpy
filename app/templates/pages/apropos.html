À propos

Cette application web est développée dans le cadre du Master 2 Technologies numériques appliquées à l'histoire de l'École nationale des chartes.

Description du projet

Epistautograpy est une application web permettant de donner accès aux lettres de Charles VIII en les classant sous un nouvel angle. Epistautograpy permet ainsi de consulter des lettres de Charles VIII par : contresignataire ; destinataire ; date.

Comment fonctionne Epistautograpy

Les différentes pages s'appuient sur les langages HTML, CSS, JavaScript et Python 3, ainsi que sur une base de données MySQL.

Comment installer Epistautograpy

Télécharger le dossier GitHub. Nous recommandons la création d'un environnement virtuel d'après le fichier requirements.txt. Ouvrir le premier dossier "epistautograpy" et lancer le fichier run.py.
